## Description of project

### Problem
Car owners often do not track their expenses systematically and forget about scheduled maintenance (oil changes, insurance, inspections). This leads to unexpected costs and a poor understanding of the real cost of car ownership.

### Solution
The application provides a centralized system to track all car-related expenses and important events. It helps users monitor spending, analyze trends, and receive timely reminders for maintenance and required payments.

### Target Audience

- **Primary audience:** car owners (18–45 years old) who want to control and optimize their expenses  
- **Secondary audience:** students, young professionals, and people interested in personal finance management

## Project Scope

### MVP (for final delivery)
- [ ] User registration and authentication (JWT)
- [ ] Add/edit/delete vehicles
- [ ] Add and delete expenses 
- [ ] View expense list with filtering
- [ ] Basic analytics (expenses by category and period)
- [ ] Notification system (maintenance, insurance reminders)
- [ ] Customise your profile (email, password, full name, country)
- [ ] 2 user roles (User / Service)

### Out of Scope (post-course)
- [ ] Map integration (finding nearby service stations)
- [ ] Expense optimization recommendations
- [ ] Export to CSV/Excel
- [ ] Mobile version / PWA
- [ ] Autoselected currency by your country
- [ ] Integration with banking APIs (auto expense tracking)
